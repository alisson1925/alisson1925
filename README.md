# 👨‍💻 Sobre Mim

Cursando **Ciência da Computação** e direcionando minha formação para a área de **Data Analytics e Business Intelligence**.

Tenho desenvolvido projetos práticos voltados à análise de dados, criação de dashboards e automação de processos utilizando **Power BI, SQL Server e Python**.

Meu objetivo é transformar dados em informações claras e úteis para apoiar a tomada de decisão em ambientes corporativos.

Atualmente, estou focado em:

- Business Intelligence  
- Análise de Dados  
- SQL aplicado a negócios  
- Power BI e Dashboards Executivos  
- Python para análise de dados  
- ETL e tratamento de dados  
- Visualização estratégica de informações  

Também possuo experiência prática com tecnologia, hardware e suporte técnico, o que contribui para uma visão mais ampla de sistemas e infraestrutura.
